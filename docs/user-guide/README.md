# User guide

Everything Granite can do from the keyboard and mouse. Granite is a
local-first Markdown knowledge base: every note is a plain `.md` file on
your disk, and every feature in this guide operates on those files
directly. Nothing ships off your device unless you opt in.

If you are brand new, start with
[Getting started](../getting-started/installation.md) first — it covers
install, first-run, and a workspace tour. This section is the
feature-by-feature manual you come back to.

## Conventions used in this guide

- `Mod` means `Cmd` on macOS and `Ctrl` on Windows/Linux. So `Mod+P`
  reads as `Cmd+P` on a Mac and `Ctrl+P` on a PC.
- `Alt` on Windows/Linux is the same as `Option` on macOS.
- A **vault** is the folder of Markdown files Granite is currently
  showing. A **tab** is one open document; a **tab group** holds
  several tabs side-by-side. The thin icon strip on the far left is
  the **ribbon**; the bar at the bottom is the **status bar**;
  collapsible panels on either side are **sidebars**.
- A **command** is any action that can be triggered from the Command
  palette (`Mod+P`). A **hotkey** is a keyboard shortcut bound to a
  command. A **notice** is a transient toast that appears in the
  bottom-right.
